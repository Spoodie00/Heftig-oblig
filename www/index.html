Most modern web applications combine HTML, CSS and JavaScript to create something called a Single Page Application (SPA). JavaScript is in some sense similar to Python in the sense that it is also a dynamically typed and interpreted language. A good starting point for learning JavaScript and Web Development in general are the Mozilla Developer Guides. Additionally, people often use some kind of a framwork to set up their application. Some popular examples are:

  • Anular
  • React
  • Vue.js
  • Svelte

Or you may use a traditional backend-driven rendering mechanism by leveraging FastAPI's HTMLResponse and htmx.
